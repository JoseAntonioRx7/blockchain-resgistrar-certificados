Registro de Certificados em Blockchain (Go + Web)
Nas últimas semanas venho estudando blockchain de forma prática e resolvi transformar isso em um projeto real.
Desenvolvi uma blockchain do zero em Go, com uma API REST e uma interface web simples em HTML/JS, capaz de:

• Gerar hash criptográfico (SHA-256) de certificados em PDF
• Registrar esses hashes em blocos imutáveis
• Minerar novos blocos com Proof of Work
• Permitir a verificação pública de certificados por hash
• Integrar backend + frontend em uma aplicação funcional

O objetivo é combater fraudes em certificados e diplomas, criando uma prova criptográfica de autenticidade que qualquer pessoa pode verificar.
